## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run start
```

## API

## Movies API

POST create a movie
api/movies

DELETE delete a movie
api/movies/:id

PATCH update a movie
api/movies/:id

POST search for movies
api/movies/search

GET get all movies with pagination
api/movies?limit=5&offset=0

## Genres API

POST create a new genre
api/genres

DELETE delete a genre
api/genres

GET get all genres
api/genres
